seqSetFilterChrom {SeqArray}R Documentation

Chromosome selection

Description

Select the variants according to the specified chromosome(s).

Usage

seqSetFilterChrom(gdsfile, include=NULL, is.num=NA)

Arguments

gdsfile

a SeqVarGDSClass object

include

NULL, or character for specified chromosome(s)

is.num

a logical variable: TRUE, chromosome code is numeric; FALSE, chromosome is not numeric

Value

None.

Author(s)

Xiuwen Zheng

See Also

seqSetFilter

Examples

# the file of GDS
gds.fn <- seqExampleFileName("gds")
# or gds.fn <- "C:/YourFolder/Your_GDS_File.gds"

f <- seqOpen(gds.fn)
seqSummary(f)


seqSetFilterChrom(f, is.num=TRUE)
seqSummary(f, "genotype", check="none")

seqSetFilterChrom(f, is.num=FALSE)
seqSummary(f, "genotype", check="none")

seqSetFilterChrom(f, 1:4)
seqSummary(f, "genotype", check="none")
table(seqGetData(f, "chromosome"))


# close the GDS file
seqClose(f)
