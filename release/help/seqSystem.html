seqSystem {SeqArray}R Documentation

Get the parameters in the GDS system

Description

Get a list of parameters in the GDS system

Usage

seqSystem()

Value

A list including

num.logical.core

the number of logical cores

compiler.flag

SIMD instructions supported by the compiler

options

list all options associated with SeqArray GDS format or packages
